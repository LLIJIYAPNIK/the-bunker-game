# UC-G3 — Голосование

## Акторы
- Игрок

## Предусловия
- Голосование открыто.
- Игрок жив.

## Основной сценарий
1. Игрок выбирает цель.
2. Система принимает голос.
3. Система сохраняет голос.
4. Система проверяет завершение голосования.

## Альтернативные сценарии
- A1: Игрок уже голосовал.
- A2: Голосование закрыто.
- A3: Игрок исключён.

## Постусловия
- Голос сохранён.
